# AbstractRaft

Status: unvalidated proof exercise.

This contributed specification is retained as a standalone, Raft-inspired TLAPS proof challenge. Its refinement relationship to Raft has not been established, so it is excluded from the canonical benchmark, default runs, dataset statistics, and leaderboard scores.

These files were included in benchmark release `v1.0.0`. They can be reconsidered for the canonical dataset if validation evidence for the abstraction is supplied.
